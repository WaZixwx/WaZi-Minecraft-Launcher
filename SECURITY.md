1. **不要**在公开场合披露漏洞
2. 发送邮件至 security@wazixwx.com
3. 在邮件中详细描述：
   * 漏洞类型
   * 复现步骤
   * 潜在影响
   * 建议的修复方案（如果有）

我们会在收到报告后的48小时内：
* 确认收到您的报告
* 评估漏洞的严重程度

* 在修复完成后通知您

# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported |
| --- | --- |
| 1.0.x | :white_check_mark: |
| < 1.0 | :x: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please follow these steps:

1. Do **NOT** disclose the vulnerability publicly
2. Send an email to security@wazixwx.com
3. Include in your email:
   * Type of vulnerability
   * Steps to reproduce
   * Potential impact
   * Suggested fix (if any)

We will within 48 hours of receiving your report:
* Acknowledge receipt of your report
* Assess the severity of the vulnerability
* Develop a fix plan
* Notify you once the fix is implemented

## Vulnerability Response Times

| Severity | Response Time | Fix Time |
| --- | --- | --- |
| Critical | Within 24 hours | Within 7 days |
| High | Within 48 hours | Within 14 days |
| Medium | Within 72 hours | Within 30 days |
| Low | Within 7 days | Within 90 days |

## Security Best Practices

To ensure secure usage of the launcher, we recommend:

1. Always download the launcher from official sources
2. Keep the launcher updated to the latest version
3. Use strong passwords for your Minecraft account
4. Never share your account credentials
5. Enable two-factor authentication when available

## Security Updates

Security updates will be released through:

* GitHub Releases
* Launcher auto-updates
* Official website announcements
